# 🔐 Advanced Password Strength Analyzer

Welcome to my Python security project! This is a dynamic script designed to evaluate the strength and complexity of passwords, helping users defend against potential brute-force vulnerabilities.

## ⚙️ How It Works
The script utilizes core programming logic and conditional flags to analyze:
- **Minimum Length:** Checks for standard baseline security.
- **Character Diversity:** Independently verifies the presence of Uppercase, Lowercase, and Numeric digits.
- **Symbol Inclusion:** Scans for vital cryptographic symbols.
- **Dynamic Scoring:** Categorizes passwords in real-time as `Weak ❌`, `Medium ⚠️`, or `Very Strong 💪`.
